# 🤖 BotBuilder - 
!On going
پلتفرم ربات‌ساز تلگرام

یه پلتفرم برای ساخت ربات‌های تلگرام بدون نیاز به کدنویسی.
کاربر فقط توکن ربات خودشو میده و ما بقیه کارا رو میکنیم.

## ✨ قابلیت‌ها

- ساخت ربات آپلودر (فایل شیرر)
- اشتراک‌گذاری فایل از طریق لینک اختصاصی
- پنل مدیریت
- در آینده: ربات فروشگاهی، پنل وب

## 🛠 تکنولوژی‌ها

- Python 3.12
- Django 5
- Django REST Framework
- aiogram 3.7 (تلگرام)
- PostgreSQL
- uvicorn (ASGI)
- ngrok (توسعه)
